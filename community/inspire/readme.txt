INSPIRE extension for GeoServer
===============================

This archive contains the INSPIRE extension for GeoServer.  The INSPIRE extension allows GeoServer to be compliant with specifications put forth by the Infrastructure for Spatial Information in the European Community (INSPIRE) directive.

Installation
------------

* Extract the contents of this archive into the <GEOSERVER_ROOT>/WEB-INF/lib directory.

* Restart GeoServer.
